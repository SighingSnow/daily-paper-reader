## 摘要
面向大规模视觉-语言-动作 (VLA) 模型的强化学习 (RL) 在计算效率和数据获取方面面临重大挑战。我们提出了 AcceRL，这是一个全异步解耦的强化学习框架，旨在通过物理隔离训练、推理和推演过程来消除同步障碍。关键在于，AcceRL 首次将即插即用、可训练的世界模型集成到分布式异步强化学习流水线中，以生成虚拟经验。在 LIBERO 基准上的实验表明，AcceRL 实现了最先进 (SOTA) 的性能。在系统层面，它在吞吐量上表现出超线性扩展能力，并具有极高的硬件利用率。在算法层面，其世界模型增强变体在复杂控制任务中实现了前所未有的样本效率和鲁棒的训练稳定性。代码已在 https://github.com/distanceLu/AcceRL 公开。

## Abstract
Reinforcement learning (RL) for large-scale Vision-Language-Action (VLA) models faces significant challenges in computational efficiency and data acquisition. We propose AcceRL, a fully asynchronous and decoupled RL framework designed to eliminate synchronization barriers by physically isolating training, inference, and rollouts. Crucially, AcceRL is the first to integrate a plug-and-play, trainable world model into a distributed asynchronous RL pipeline to generate virtual experiences. Experiments on the LIBERO~\cite{liu2023libero} benchmark demonstrate that AcceRL achieves state-of-the-art (SOTA) performance. Systematically, it exhibits super-linear scaling in throughput and highly efficient hardware utilization. Algorithmically, the world-model-augmented variant delivers unprecedented sample efficiency and robust training stability in complex control tasks. Code is publicly available at https://github.com/distanceLu/AcceRL.

## 2. 论文提出的方法论
本文提出了 AcceRL，一个全异步、解耦的分布式强化学习框架，并首次集成了可训练的世界模型。

### 核心思想
*   **全异步解耦架构**：将训练、推理和环境交互物理隔离到独立的异步流中，消除全局同步屏障。
*   **世界模型增强**：引入即插即用的世界模型进行“想象推演”，用高保真的虚拟经验替代昂贵的真实交互。

### 关键技术细节
#### A. 双层异步机制
1.  **宏观异步（训练与推理解耦）**：
    *   采用非阻塞循环流水线，Rollout Workers 持续产生数据并存入分布式 Replay Buffer，Trainer Workers 独立采样更新。
    *   利用 ZeRO 优化技术分割优化器状态，通过 NCCL 广播同步参数，实现超线性吞吐量扩展。
2.  **微观异步（交互与推理解耦）**：
    *   提出“推理即服务”范式。Rollout Workers 仅负责环境交互，将观测发送至集中的 Inference Pool 进行批量推理。
    *   **动态批处理触发**：设定触发条件 $Trigger = (|Q| \ge B) \lor (t_{now} - t_{first} \ge T_{max})$，平衡吞吐量与延迟。

#### B. 解决策略滞后
针对异步架构中行为策略与当前策略不一致的问题，提出三项措施：
*   **价值重计算**：使用最新参数重新计算价值估计 $V^{new}_t$，确保优势估计的准确性。
*   **全局优势归一化**：通过 AllReduce 聚合全局统计量，消除局部偏差。
*   **梯度校准（GIPO）**：引入高斯重要性采样策略优化，替代 PPO 的硬裁剪，通过软惩罚平滑极端重要性比率，公式为 $\omega(\bar{\rho}_t; \sigma) = \exp(-\frac{1}{2}(\frac{\log \bar{\rho}_t}{\sigma})^2)$。

#### C. 世界模型集成
*   **观测模型 ($M_{obs}$)**：基于扩散模型（DIAMOND 架构），生成高保真的下一帧图像。
*   **奖励模型 ($M_{reward}$)**：基于 OpenVLA-OFT 修改的二分类器，评估成功概率。
*   **潜在奖励机制**：定义奖励为相邻状态成功概率之差 $\hat{r}_\tau = M_{reward}(\hat{o}_{\tau+1}) - M_{reward}(\hat{o}_\tau)$，提供稠密信号且保持策略不变性。
*   **交替采样策略**：在真实环境交互与想象推演之间交替，利用真实数据训练世界模型，利用虚拟数据优化策略。

#### D. VLA 模型优化
*   **词表精简**：裁剪语言模型头，仅保留动作 Token，减少显存占用并降低噪声。
*   **Token 级优化**：将动作序列视为独立 Token 进行优化，避免 Chunk-level 联合概率导致的数值不稳定。
