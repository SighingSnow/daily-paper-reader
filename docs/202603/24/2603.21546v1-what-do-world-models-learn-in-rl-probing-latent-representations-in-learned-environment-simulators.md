## 摘要
世界模型从经验中学习模拟环境动力学，从而实现样本高效的强化学习。但这些模型内部实际上表征了什么？我们将可解释性技术——包括线性和非线性探针、因果干预和注意力分析——应用于两个架构迥异的世界模型：IRIS（离散token Transformer）和DIAMOND（连续扩散UNet），这两个模型均在Atari Breakout和Pong游戏上进行了训练。利用线性探针，我们发现这两个模型都发展出了游戏状态变量（物体位置、分数）的线性可解码表征，而MLP探针得出的R^2仅略高，证实了这些表征是近似线性的。因果干预——沿探针推导的方向平移隐藏状态——会在模型预测中产生相关的变化，这提供了证据表明这些表征是被功能性利用的，而不仅仅是相关关系。对IRIS注意力头的分析揭示了空间特化现象：特定的头优先关注与游戏物体重叠的token。多基线token消融实验一致表明，包含物体的token具有不成比例的重要性。我们的发现提供了可解释性证据，表明学习到的世界模型在两种游戏和两种架构中，都发展出了环境状态的结构化、近似线性的内部表征。

## Abstract
World models learn to simulate environment dynamics from experience, enabling sample-efficient reinforcement learning. But what do these models actually represent internally? We apply interpretability techniques--including linear and nonlinear probing, causal interventions, and attention analysis--to two architecturally distinct world models: IRIS (discrete token transformer) and DIAMOND (continuous diffusion UNet), trained on Atari Breakout and Pong. Using linear probes, we find that both models develop linearly decodable representations of game state variables (object positions, scores), with MLP probes yielding only marginally higher R^2, confirming that these representations are approximately linear. Causal interventions--shifting hidden states along probe-derived directions--produce correlated changes in model predictions, providing evidence that representations are functionally used rather than merely correlated. Analysis of IRIS attention heads reveals spatial specialization: specific heads attend preferentially to tokens overlapping with game objects. Multi-baseline token ablation experiments consistently identify object-containing tokens as disproportionately important. Our findings provide interpretability evidence that learned world models develop structured, approximately linear internal representations of environment state across two games and two architectures.

## 2. 论文提出的方法论
本研究主要采用三种可解释性技术来探测和分析模型的潜在表征：

*   **线性与非线性探测**：
    *   **核心思想**：通过训练探针分类器从模型的隐藏层提取表征，以此预测游戏状态变量（如球的位置、挡板位置、分数）。
    *   **技术细节**：提取冻结的模型表征，分别训练岭回归作为线性探针和 2 层 MLP（256→128→1, ReLU）作为非线性探针。使用 5 折交叉验证的 $R^2$ 作为评价指标。
    *   **关键指标**：定义选择性差距 $\Delta = R^2_{MLP} - R^2_{linear}$。若 $\Delta$ 很小，说明表征是近似线性的。

*   **因果干预**：
    *   **核心思想**：验证探针发现的方向是否具有因果效应，而非仅仅是统计相关。
    *   **算法流程**：沿探针权重的方向对隐藏状态进行平移干预，公式为 $h' = h + \alpha \cdot \hat{w}$，其中 $\hat{w}$ 是归一化的岭回归探针权重。随后测量模型预测输出（下一帧 token 或像素）的变化（KL 散度）。

*   **注意力分析与 Token 消融**：
    *   针对 IRIS 模型的 Transformer 结构，计算注意力头的空间选择性，分析其是否关注特定游戏对象。
    *   进行多基线 Token 消融（零向量、均值、随机码本向量），通过预测破坏程度（KL 散度）评估各 Token 的重要性。

## 6. 论文的主要结论与发现
*   **近似线性表征的存在**：两种模型均发展出了游戏状态变量的线性可解码表征。MLP 探针相比线性探针的提升微乎其微（IRIS 的 $\Delta \le 0.06$），证实了表征的近似线性。
*   **架构差异导致的表征分布差异**：
    *   **IRIS**：VQ-VAE 编码器已具备很强的线性表征能力，Transformer 层主要起保持和传递作用，暗示了“分词器负责空间编码，Transformer 负责时序预测”的分工。
    *   **DIAMOND**：表征能力呈现倒 V 型，瓶颈层压缩了最抽象的状态信息，解码器层通过跳跃连接非线性地恢复细节。
*   **因果有效性**：因果干预实验表明，沿探针方向干预隐藏状态会显著改变模型预测（$r > 0.95$），证明这些线性表征被模型功能性地使用。
*   **注意力专门化**：IRIS 的特定注意力头表现出对特定空间区域（如游戏对象所在位置）的偏好，存在分工机制。

## 7. 优点
*   **跨架构验证**：不仅研究了常见的 Transformer 架构，还引入了扩散模型架构，使得结论不局限于单一网络结构，更具说服力。
*   **因果层面的深入分析**：超越了单纯的“探测”相关性，引入激活修补进行因果干预，有力证明了内部表征确实在驱动模型预测。
*   **揭示模型分工**：通过层级探测，清晰揭示了 IRIS 和 DIAMOND 内部不同模块（如 Tokenizer 与 Transformer，Encoder 与 Bottleneck）的功能分工。

## 8. 不足与局限
*   **环境局限**：实验仅限于 2D Atari 游戏，场景相对简单，结论能否推广到 3D 或更复杂的现实环境尚不可知。
*   **探测方法的局限**：单帧探测可能遗漏了 Transformer 编码的时序动态信息；因果干预仅沿单一方向进行，可能过于粗糙，无法捕捉复杂的非线性交互。
*   **数据规模**：探测仅使用了 10,000 帧数据，虽然对于简单游戏可能足够，但在更复杂场景下的统计显著性可能需要更多数据支持。
